# Financial Analytics Dashboard

## Project Overview

This project analyzes financial data using Excel, PostgreSQL, SQL, and Power BI.

The objective is to identify growth trends, financial performance, and business insights through data analysis and visualization.

## Tools Used

* Excel
* PostgreSQL
* SQL
* Power BI
* GitHub

## Project Workflow

1. Data Cleaning
2. Data Import into PostgreSQL
3. SQL Analysis
4. Power BI Dashboard Development
5. Business Insights
